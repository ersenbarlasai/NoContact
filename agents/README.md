# Agents

This folder stores the project's in-repo agent system. Each agent has:
- `RULES.md`: non-negotiable operating rules.
- `MEMORY.md`: what the agent should remember between sessions.
- Optional future `SKILLS.md`: reusable workflows the agent can improve over time.

Before major work, read:
1. `docs/PROJECT_BRIEF.md`
2. `docs/agents/AGENT_OPERATING_SYSTEM.md`
3. The relevant agent `RULES.md` and `MEMORY.md`
