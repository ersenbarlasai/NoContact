# UX/UI Guardian Memory

- Canonical Design System: **Still** ("The Quiet Companion").
- Desired feel: calm, grounded, private, premium, trustworthy.
- Visual Language: Warm Minimalism, Visual Silence.
- Colors: Sage (Action), Terracotta (Human/SOS), Neutral/Sand surfaces.
- Typography: Literata (Headlines/Supportive), Manrope (Body/UI).
- SOS Strategy: Visible terracotta-based action, not aggressive red.
- First screen should show counter, SOS, mood check-in, next action, progress insight.
- Stitch will be used for frontend design exploration.
