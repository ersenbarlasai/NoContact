# Design System

# Product Feel
Calm, grounded, private, warm, and decisive. The user may arrive anxious or ashamed; the interface should lower emotional temperature and give one clear next action.

# Visual Direction
- Avoid loud romance visuals, heartbreak cliches, and dark dramatic gradients.
- Use soft contrast, clean spacing, and stable layouts.
- The app should feel more like a confidential recovery companion than a dating app.
- First screen must be useful immediately.

# Typography
- Use a highly readable sans-serif.
- No negative letter spacing.
- Avoid viewport-based font scaling.
- Large type only on the main dashboard counter and onboarding headline.

# Components
- Persistent SOS button: icon + short label, reachable from home and bottom nav.
- Mood selector: accessible segmented control or icon buttons with labels.
- Urge timer: stable circular timer with breathing guidance.
- Insight card: short, concrete, no walls of text.
- Paywall: transparent feature comparison, no dark patterns.
- Safety notice: concise and visible around AI features.

# Stitch Prompt Seed
Design a mobile app dashboard for a private breakup recovery companion. Turkish-first UI. The screen shows a no-contact day counter, one prominent SOS button, a daily mood check-in, a next recommended healing action, and a subtle weekly progress insight. Visual style: calm, trustworthy, premium, soft ivory background, deep ink text, mint and coral accents, accessible contrast, no dating-app cliches, no dramatic heartbreak imagery. Use iOS and Android friendly mobile patterns.
