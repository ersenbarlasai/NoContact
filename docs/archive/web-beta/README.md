# Web Beta Archive

This directory contains documentation and configurations for the temporary Web Beta of Still, which was hosted on Vercel.

The Web Beta was conducted to gather early user feedback before the official mobile release. It has now been closed, and the application has returned to a mobile-only focus.

## Contents
- `WEB_BETA_VERCEL_PLAN.md`: The original plan for the web beta deployment.
- `VERCEL_WEB_BETA_DEPLOYMENT_CHECKLIST.md`: Checklist used for the Vercel deployment.

*Archived on: 2026-05-11*
