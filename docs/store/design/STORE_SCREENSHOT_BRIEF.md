# Store Screenshot Brief - NoContact MVP

## 1. Specifications
### iOS Requirements
- **6.7-inch (iPhone 14/15 Pro Max):** 1290 x 2796 pixels
- **6.5-inch (iPhone 11 Pro Max):** 1242 x 2688 pixels
- **5.5-inch (iPhone 8 Plus):** 1242 x 2208 pixels

### Android Requirements
- **Standard Phone:** 1080 x 1920 pixels (or equivalent 16:9 / 18:9 ratio)

### Formatting Guidance
- **Safe Area:** Keep text and key elements away from the top/bottom edges (notches, home indicators).
- **Phone Frame:** Use a minimal, flat device frame or a subtle clay mockup. Do not use hyper-realistic glossy renders.
- **Background Color:** Use Cream (`#FAF9F6`) or a very soft Sage (`#E8EFEA`) for the background to maintain the "Nosto" aesthetic.
- **Typography:** Inter or Manrope. Bold for headlines, regular for captions. Dark Charcoal color.
- **Overlay Text Placement:** Top 20% of the image. The device frame sits below the text.

## 2. Screenshot Composition

### Screenshot 1: Home Dashboard
- **Screen:** Home / Ana Ekran
- **Visual Focus:** Clean dashboard, No Contact counter, calm greeting.
- **Headline:** Sınırlarını Koruman İçin
- **Caption:** Kendi iyileşme hızına saygı duyan, yargısız ve sessiz bir alan.
- **Mock Data:** "14 Gündür İletişimde Değilsin." "Bugün kendini nasıl hissediyorsun?"

### Screenshot 2: SOS Mode
- **Screen:** SOS / Acil Destek
- **Visual Focus:** Breathing circle or "Yaz ve Sil" input.
- **Headline:** Ani İstekleri Yönet
- **Caption:** Bir mesaj atmadan önce durakla, hislerini yaz ve güvenle bırak.
- **Mock Data:** "Şu an yazmak istiyorum ama biliyorum ki..."

### Screenshot 3: Unsent Letters Vault
- **Screen:** Mektup Kasası
- **Visual Focus:** List of letters with a lock/privacy icon.
- **Headline:** Söylenmemiş Sözler
- **Caption:** Göndermekten vazgeçtiğin mektupları cihazında şifreli olarak sakla.
- **Mock Data:** "Sana Söyleyemediklerim", "Bugün Hissettiklerim".

### Screenshot 4: Mood Journal
- **Screen:** Duygu Günlüğü
- **Visual Focus:** Low-arousal mood slider and note input.
- **Headline:** Kendini Anlamaya Başla
- **Caption:** Her gün nasıl hissettiğini kaydet ve duygusal dalgalanmalarını izle.
- **Mock Data:** Seçilen: "Huzurlu", Not: "Bugün düne göre biraz daha sakinim."

### Screenshot 5: Quiet Growth Insights
- **Screen:** Sessiz Gelişim
- **Visual Focus:** Calm progress summary, no aggressive streaks.
- **Headline:** Kendi Ritminde Geliş
- **Caption:** İyileşme sürecini sessiz ve baskısız bir şekilde takip et.
- **Mock Data:** "Yönetilen İstek: 5", "Milestone: İlk Adım".

### Screenshot 6: Daily Rhythm
- **Screen:** Günlük Ritim
- **Visual Focus:** Time picker and simple notification toggle.
- **Headline:** Nazik Hatırlatmalar
- **Caption:** Sadece istediğin zamanlarda, kendine dönmen için küçük bir hatırlatma.
- **Mock Data:** Açık (ON), Saat: "20:30".

### Screenshot 7: Privacy Reassurance
- **Screen:** Biyometrik Kilit
- **Visual Focus:** Face ID/Fingerprint prompt over blurred background.
- **Headline:** Tamamen Sana Ait
- **Caption:** Notların ve mektupların sadece bu cihazda kalır.
- **Mock Data:** Biyometrik doğrulama ekranı, "Şifreli" ikonu.

## 3. UI Text Localization (Crucial)
When mocking up the screens inside the device frames, ALL in-app UI text must be in Turkish. Do not use English placeholders.

Use the following translations for common UI elements:
- "Morning, Alex" -> **"Bugün kendini seçtin"**
- "Day 14 of choosing yourself" -> **"14 gündür temas yok"**
- "Unsent Letters" -> **"İsimsiz Mektuplar"**
- "My Journey" -> **"İyileşme Yolum"**
- "Begin Session" -> **"Seansa Başla"**
- "Daily Rhythm" -> **"Günlük Ritim"**
- "Rhythm & Privacy" -> **"Ritim ve Gizlilik"**
